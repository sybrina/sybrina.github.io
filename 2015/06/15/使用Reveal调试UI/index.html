使用Reveal调试UI

发表于 2015-06-15   |   分类于 iOS开发

调试UI是一件费时费力的活，如果你的UI都是用代码写的话，偶尔会莫名其妙的出现UI异常，检查代码又检查不出错误，这时候你就需要Reveal这款UI调试工具来帮你检查UI。

如何使用？

首先需要在(Reveal官网)[http://revealapp.com] 或者AppStore下载该应用，该应用有30天的免费无限制试用。下载完毕之后安装即可。

如果你使用CocoaPods来管理第三方库，你可以在项目的podfile中添加

并执行pod install来集成它。Reveal-iOS-SDK属于私有库，如果你不使用:configurations => ['Debug']选项，在你的App打包上传到AppStore时会出现Your app contains non-public API usage错误，所以如果需要上架，请务必使用Debug选项。

如果你没有使用CocoaPods，你需要在Reveal的Help-Show Reveal Library in Finder中找到Reveal.framework并添加到你的项目中即可。

接下来运行你的项目，并在Reveal选择你的项目，就可以看见当前页面的UI层级显示了，你可以很清晰的看见每个UI的逻辑关系，并可以在右侧的面板（和Xcode中得右侧Utilities面板功能几乎一样）中更改该视图的相关属性。并可以查看修改的实时效果。缺点是在Reveal中得修改不能实时的修改工程代码，但是这对于UI调试来说确实能节省不少时间。
