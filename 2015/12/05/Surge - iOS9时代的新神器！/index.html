Surge - iOS9时代的新神器！

发表于 2015-12-05   |   分类于 软件

2016-01-22 更新

Surge 上架的时候是68元RMB，不过 Surge 上架仅仅5周后作者就被约去”喝茶”，之后在全区下架，作者在 twitter 上说 Surge 会不定期的上线以便修复现有 bug，上线的时候价格会调成 $999，前几天 Surge 上线过一次，不过价格是699RMB，幸好我入手的早😝。

Surge 下架之后，开发者移除了自己配置服务器部分的功能，重新上架了一款 Replica 的应用，专门用来进行网络调试，已购买 Surge 用户可以直接解锁 Replica Pro 版本。另外，Surge for mac 目前在内测阶段，也将在不久后上架，希望能代替 GoAgentX + Proxifier 组合，很值得期待！
开发者Twitter：Yachen Liu

一直以来，在iOS设备上可以全局科学上网的途径除了VPN外，就是使用shadowsocks了。然而，VPN的协议特征比较明显，非常容易被某墙针对，而且VPN无法智能科学上网，诸如微信，微博之类的国内服务在开启VPN之后延迟就比较大。如果你的设备没有越狱，shadowsocks在iOS也有客户端可以使用，但是仅能通过shadowsocks自带的浏览器访问墙外内容，其它应用也无法使用shadowsocks，除非用户对设备进行越狱，而越狱带来的数据安全风险也是比较大的。

终于，iOS9带来了一个网络新特性Network Extension，基于此开发的Surge可以在不越狱的情况下实现很多功能。比如观察各种 APP 的联网情况、监视 HTTP / TCP 请求、网络调试、流量跟踪、流量统计、屏蔽指定网址、过滤广告，以及实现那个非常重要的功能等等。需要有专业知识背景的用户才可使用。一旦你按自己的需求配置好 Surge 以及服务器之后，你会发现 iOS 的世界将瞬间变得畅通无阻了！

目前Surge已上架AppleStore，售价68元人民币，Surge AppleStore 传送门，并且开发者称3个月内不会降价。对于我们来说，这是有和没有的区别，特别是对于科学上网已经算是刚需的人群来说，Surge绝对称的上是神器！

如何使用？

Surge使用是有一定门槛的，如果从 0 开始自己去配置规则，相信能自己搞定的人也没有几个，好在网上已经有现成的配置教程，按照下面的步骤来，小白也可以很容易进行配置。

小众软件：http://www.appinn.com/surge-for-ios/

简书小白教程：http://www.jianshu.com/p/de1eb844915d

如果你想了解Surge的实现原理，github上有个开源项目NetworkEye也主要用到了Network Extension特性，可以作为参考。
