把后台应用关进冰箱吧！

发表于 2016-03-21   |   分类于 折腾笔记

妹子的手机是一部 Nexus 6，一般用上一天就会很卡，之前我的解决办法是重启，重启完了会好点，但是从根本上解决不了问题，而且这部手机耗电异常严重，我每次看见几乎都插着充电器 😥

最近同事安利了一款国产的应用 Ice Box ，专治安卓后台应用引起的卡顿问题，使用该应用需要手机先 Root，一般国产的手机都比较容易Root，各种 Root 精灵、Root 大师，傻瓜式操作，一键就能 Root。可惜，并不支持 Nexus 6 😂😂😂。

趁着这段时间闲点，抽空把手机数据都备份好了。从 inexus 找到了 Root 方法，步骤稍微有点麻烦，而且 Root 失败的话变砖风险比较大。

Root 步骤

  • 升级系统到 Android 6.0 或 6.0.1；
  • 安装 adb 和 fastboot 驱动；
  • 解锁 bootloader；
  • 刷入第三方 Recovery，比如我刷的是 twrp；
  • 利用 twrp 刷入 supersu 实现修改内核进行 Root。

需要注意的地方

  • 解锁会失去所有数据，请提前做好备份，本次 Root 环境为 Windows 10；
  • Nexus 6 解锁前需要在设置里 开启oem解锁；
  • Root 过程中遇到点问题，在刷入了 twrp 后，从 bootloader 进入 twrp 遇到了 “无命令” 的提示，显然是刷入 twrp 失败，在 bootloader 模式下清空数据后再次刷入成功；

其它问题链接：

  • 解决刷机出现adb或者fastboot不是内部或者外部命令
  • 解锁 bootloader 教程
  • Root 教程

资源

  • ADB 和 fastboot 驱动（64位）

总之，本次 Root 过程比较顺利，基本没遇到什么大问题，冰箱 Ice Box 这款应用也非常给力，6儿子再也不卡了，😄😄😄
